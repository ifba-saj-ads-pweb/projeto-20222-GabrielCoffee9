Programação WEB - Projeto do Semestre

Sobre o sistema:

O projeto do semestre de 2022.2 da disciplina programação web é um sistema de gerenciamento de mesas e pedidos em um estabelecimento, como por exemplo um restaurante.

Objetivos do projeto:

Utilizar Html, CSS, JavaScript e outras ferramentas para construir um website limpo e funcional.

Simplificar e agilizar a interação do cliente ao entrar em um estabelecimento.
